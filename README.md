# 多個定時鬧鐘 (Multiple Countdown Alarms)

一個基於 Web 的輕量級倒數計時工具，允許使用者在同一頁面上同時管理與執行多個獨立的定時鬧鐘。支援自訂音檔、內建頻率合成音效（Beep）以及文字轉語音（TTS）功能。

## ✨ 功能特點

- **動態產生鬧鐘**：可自由設定鬧鐘數量（支援 1 至 24 個），介面會自動產生對應區塊。
- **獨立控制**：每個鬧鐘擁有獨立的倒數時間、備註與控制按鈕（開始/結束）。
- **多樣化提醒方式**：
    - **MP3 音檔**：支援上傳自訂音效（若無檔案，系統會嘗試播放 `test-beep.mp3`）。
    - **內建合成音效**：使用 Web Audio API 產生蜂鳴聲，無需外部檔案。
    - **語音朗讀 (TTS)**：支援瀏覽器原生的語音合成，倒數結束時可朗讀「備註」內容。
- **高度客製化蜂鳴聲**：
    - 波形選擇 (Sine, Square, Sawtooth, Triangle)
    - 音高頻率 (Hz)
    - 聲音長度與間隔 (ms)
    - 音量與重複次數
- **即時狀態顯示**：清楚顯示「按下時間」、「預計觸發時間」以及「剩餘時間」。

## 🚀 使用說明

1. **設定數量**：在頂部輸入欄位設定需要的鬧鐘數量（預設 2 個）。
2. **輸入備註**：在鬧鐘標題旁輸入備註文字（例如：「泡麵」、「番茄鐘工作」），這將作為語音朗讀的內容。
3. **設定時間**：輸入倒數的分鐘數與秒數。
4. **選擇音效**：
    - 你可以上傳自己的 `.mp3` 檔案(網頁版只能用預設mp3)。
    - 或者勾選「使用語音提醒」來聽取備註內容。
    - 若未設定，將使用內建的蜂鳴聲。
5. **開始倒數**：點擊「開始倒數」，系統會計算觸發時間並開始計時。

## 🛠️ 進階設定 (內建蜂鳴器)

每個鬧鐘都可以微調內建蜂鳴器的聲音特性，這對於沒有準備 MP3 檔案的使用者非常方便：

- **波形 (Waveform)**：決定音色（如 Square 方波聲音較銳利，Sine 正弦波較圓潤）。
- **音高 (Frequency)**：調整聲音的高低（預設 1200Hz）。
- **長度與間隔**：控制「嗶—嗶—」聲的節奏。
- **試聽功能**：在設定面板下方可隨時試聽語音或蜂鳴聲效果。
